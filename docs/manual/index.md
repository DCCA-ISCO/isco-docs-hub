# Manually Maintained Docs

Documents in this section are maintained directly in `isco-docs-hub` rather than pulled from an external source. Use this for docs from SharePoint, local files, or any source without a GitHub repository.

## How to add a document

1. Create a Markdown file under `docs/manual/` in this repo.
2. Add it to the `nav` section of `mkdocs.yml`.
3. Open a PR — no sync needed, changes go live on merge.

For SharePoint pages: copy the content into Markdown format, or export to Word and convert with [Pandoc](https://pandoc.org/).
